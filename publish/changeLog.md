### 新增

- 新增歌曲下载与下载管理，支持音质、保存路径、歌手/专辑目录规则及最多 50 个并行任务
- 新增播放页、歌曲菜单及歌单批量下载入口
- 新增 WebDAV 完整备份与恢复，本地备份与 WebDAV 统一覆盖歌单、音源、设置及屏蔽规则
- 新增歌单专辑封面显示开关
- 新增当前播放队列面板，支持上滑打开、下滑关闭、拖动排序、清空及快速移除
- 新增删除已下载记录时选择是否同时删除本地音频和歌词文件
- 新增 GitHub Actions Android 打包流程

### 优化

- 已下载歌曲优先从本地文件播放，不再重复请求网络音源
- 优化下载链接刷新、音源请求头、失效存储目录回退及下载任务持久化逻辑
- 优化歌单批量选择、全选、收藏、播放及下载交互
- 统一搜索、排行榜、歌单等页面的音源选择与顶部操作栏样式
- 优化列表封面缓存和滚动体验，减少来回滑动时的重复刷新
- 重组设置页分类，完善下载、WebDAV、外观和 About 二级页面
- 使用扁平化自适应应用图标
- 应用 ID 调整为 `com.github.toside.music`，可与原版同时安装
- 最低支持 Android 11，目标 Android 16
- 升级 React Native、Navigation、Android SDK、NDK、Gradle 及相关稳定依赖，保持新架构启用

### 修复

- 修复部分网易云歌单、排行榜及歌曲详情解析异常
- 修复下载完成后旧失败任务仍显示在“正在下载”的问题
- 修复取消、失败或移除下载任务后遗留半成品音频、歌词和封面缓存的问题
- 修复部分音源下载链接过期、请求头不兼容导致下载失败的问题
- 修复字体大小为零导致的文字渲染崩溃
- 修复 Debug 安装包缺少 JavaScript Bundle 的问题
- 修复部分原生库未适配 16KB 内存页的问题
- 修复底部播放器被系统手势区遮挡及弹窗滑动残影问题
- 修复正式包中 React Native 弹窗类被 R8 改名导致的启动崩溃
